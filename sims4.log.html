Sims 4 cheats

Please read carefully the instructions before proceeding.

Pressing Ctrl + Shift + C will open an input box at the upper left of the screen which allows you to type the provided commands below.

Pressing Enter will activate the typed command. Pressing Ctrl + Shift + C again if you wish to close the input box.

It is mandatory to input the "testingcheats on" command before entering any commands, and please study the effect of the commands before entering them.

modifyrelationship W X Y Z A B_C
Note: This command is a bit confusing and case sensitive, it is advised to check your inputs thoroughly.

Replace the following variables with the corresponding value that is being asked for the command to work properly, also remember to be precise when entering the required values.

Variable W requires your sim's first name.
Variable X requires your sim's last name.
Variable Y requires the target sim's first name.
Variable Z requires the target sim's last name.
Variable A requires the amount of affection to the target sim.
Variable B requires a label, either Frienship or Romance.
Variable C stands for command word "Main"

E.g. for Friendship label:
modifyrelationship Jim Bob Julia Smith 50 Friendship_Main

E.g. for Romantic label:
modifyrelationship Jim Bob Julia Smith 100 Romance_Main
